# Initial Concept
一个基于 Three.js 和 MediaPipe Hands 的 3D 圣诞树互动 Web 应用，旨在为用户提供沉浸式的节日氛围体验。

# 产品定义

## 目标用户
*   **普通 Web 用户**：寻求节日趣味装饰、互动娱乐或独特视觉体验的互联网用户。

## 核心功能
*   **程序化圣诞树**: 基于粒子系统生成的 3D 圣诞树，支持多种色彩主题。
*   **动态人像相册**: 支持挂载多张（>3张）3:4 比例的人像照片。系统采用智能布局算法，自动在树表面分布相框，并确保相框贴合树体坡度。
*   **实时手势追踪**: 集成 MediaPipe Hands，支持通过摄像头手势旋转、缩放和控制树的效果。

## 运行环境
*   **桌面端浏览器**：针对大屏幕和网络摄像头进行优化，提供最完整的手势交互体验。

## 视觉风格
*   **延续既有风格**：在整体保持节日氛围的基础上，严格保留现有的主题配色设计，特别是**粉白主题**不作大幅改动。

## 项目愿景
*   **病毒式传播**：通过允许用户创作并分享独一无二的圣诞树截图，推动应用在社交媒体上的传播。
