# Track Specification: 动态人像相册与智能布局

## 1. 概述 (Overview)
本任务旨在升级圣诞树的装饰系统，将现有的少量预设图标替换为支持多张（>3张）的人像照片，并实现一种基于算法的“智能布局”系统。相框将自动在圣诞树表面寻找合适的位置进行挂载，无需人工指定坐标。

## 2. 功能需求 (Functional Requirements)
- **多照片支持**：支持从配置文件加载任意数量的照片（建议 5-15 张以保证性能）。
- **3:4 肖像相框**：修改 3D 相框的几何模型（Geometry），从 1:1 调整为 3:4 的经典人像比例。
- **智能布局算法 (Random Poisson)**：
    - 算法将自动在圣诞树的锥形表面生成分布点。
    - 实现简单的冲突检测或排斥逻辑，确保相框之间不会互相穿模或重叠过深。
    - 相框的朝向将自动指向树轴外部（Normal direction），确保自然贴合在树枝表面。
- **材质优化**：确保新的人像照片在 3D 环境中清晰且亮度适中。

## 3. 非功能需求 (Non-Functional Requirements)
- **性能优化**：由于增加了 Mesh 数量，需确保在多张照片下依然保持 60FPS 的渲染速度。
- **易扩展性**：后续只需在数组中增加图片路径，布局应自动重新计算。

## 4. 验收标准 (Acceptance Criteria)
- [ ] 树上挂载的照片数量可以轻松超过 3 张。
- [ ] 所有相框均为 3:4 的竖版人像比例。
- [ ] 每次刷新页面或重新加载时，相框能分布在树的不同位置，且没有明显的重叠。
- [ ] 相框的倾斜角度应跟随树木表面的坡度，看起来是“挂”在树上的。

## 5. 出发点/非目标 (Out of Scope)
- 用户前端上传界面（本次仅实现代码/预设加载）。
- 视频支持。
