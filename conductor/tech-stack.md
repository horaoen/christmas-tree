# 技术栈

## 前端核心
*   **JavaScript (ES Modules)**: 采用原生 ES6+ 语法，利用模块化组织代码结构。
*   **Three.js**: 作为 3D 渲染的核心引擎，处理场景、相机、光照以及基于 BufferGeometry 的粒子系统。
*   **程序化布局算法**: 实现了基于随机排斥（Poisson-like）的 3D 空间点生成算法，用于装饰物的智能分布。
*   **GLSL (Shaders)**: 自定义顶点与片元着色器，实现圣诞树的程序化生成、五角星的 SDF 渲染以及动态落雪效果。

## 交互与追踪
*   **MediaPipe Hands**: 基于 Web 摄像头的实时手势识别，提取 21 个手部关键点数据。

## 构建与开发
*   **Vite**: 用于前端项目的构建和热更新，提供高效的开发环境。

## 资源
*   **CSS**: 原生 CSS 处理基础布局。
*   **PNG/Images**: 用于粒子纹理（如 `snow.png`）。
